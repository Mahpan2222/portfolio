Ongoing Autoliv, Sweden · Degree Thesis (Paid)

Posture Estimation for Motorcycle Riders Using Multi-IMU Systems

Mahdi Panahi · Halmstad University / Autoliv

Developing a real-time posture estimation pipeline using multiple IMU sensors mounted on a motorcycle rider. The system combines ESP32 microcontrollers running microROS, MediaPipe for ground-truth pose validation, and sensor fusion via Extended Kalman Filter, Madgwick, and Mahony algorithms.
